# Build

JesBrains의 CLion에서 실행함

- Clone OpenGL project
```commandline
$ git clone git@github.com:habijung/learn-opengl.git
$ git submodule update --init --recursive --remote
```
- CLion에서 프로젝트 폴더 열기
- Build CMake
- Run Project
